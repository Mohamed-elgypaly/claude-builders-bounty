# Next.js + SQLite SaaS Template

This is a production-ready `CLAUDE.md` template for Next.js 15 SaaS projects using SQLite.

## How to use
1. Copy the `CLAUDE.md` from the `templates/nextjs-sqlite-saas/` directory to the root of your project.
2. Claude Code will automatically detect it and follow the project conventions.

## Features
- Optimized for Next.js 15 App Router.
- Best practices for SQLite and Drizzle ORM.
- Clear folder structure and naming conventions.
- Performance tips (WAL mode).
